-> Contexto: Definições de W3C

- O que é Webservices?
R = Por ser bastante complexo a gente pode simplificar que ele é um componente de
software reutilizavel que é exposto pelo protocolo http que pode ser consumido em 
varias aplicações como postman, browser etc..

- Quais são as principais arquiteturas?
R = As principais arquiteturas são o SOAP e REST, onde no começo o soap era bem
mais utilizado mas atualmente o REST domina a maioria dos servicos

- Quais são as diferenças dessas arquiteturas?
R = 
	- SOAP:
		= Protocolo de troca de mensagens em XML
		= Não retorna um resultado facilmente legivel para humanos
		= Performace não é das melhores
		= Invoca serviços atraves de chamadas de RPC
	- REST:
		= Um estilo arquitetural
		= Usa XML,JSON etc para enviar e receber dados
		= Chama serviços via URL PATH
		= Resultado legivel por humanos por ser basicamente JSON E XML
